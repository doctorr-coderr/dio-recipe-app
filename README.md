# RecipeSnap

A minimalist Flutter recipe application built using **Flutter Bloc** and **Dio**.

The app consumes data from the DummyJSON Recipes API and demonstrates full CRUD operations:

- GET
- POST
- PUT
- PATCH
- DELETE
